Il y aura des moments où vous voudrez partager des nouvelles,
une mise à jour de statut sur un projet,
ou peut-être voudriez-vous une page
où les gens peuvent en savoir plus sur un événement ou un programme.
Dans cette leçon,
je vais vous montrer comment créer un site de communication SharePoint
et vous montrer comment utiliser ces sites avec votre équipe.
À partir de ma page de démarrage SharePoint,
je sélectionne Créer un site.
Je veux partager des informations et engager d'autres spectateurs.

Donc, je vais choisir d'en faire un site de communication.
Je nomme ce site « La conférence sur les compétences ».
Pour l'instant, le laisse la description vide.
Je sélectionne la langue et je clique sur Terminer.
Après avoir cliqué sur Terminer,
j'arrive sur mon nouveau site de communication.

Et à partir de là, je peux modifier le contenu.
Je veux que tout le monde soit enthousiasmé
par la prochaine conférence sur les compétences.
Donc, je vais utiliser ce site pour partager des informations
afin de créer une anticipation pour l'événement.
Ce site n'a pour l'instant qu'une seule page.
J'aime le look de cette page d'accueil,
mais je veux voir si les modèles m'offriront d'autres options.
Donc, je vais aller dans le coin supérieur droit
et sélectionner les paramètres.

J'ai plusieurs options à choisir ici.
La première chose que je veux faire est d'appliquer un modèle de site
pour aider à modifier la mise en page globale de mon site.
Donc, je vais sélectionner Appliquer un modèle de site.
Une fois que la fenêtre Sélectionner un modèle de site s'ouvre,
vous avez le choix entre plusieurs modèles.
Si vous survolez un modèle et sélectionnez les détails,
vous obtiendrez une image plus claire de la disposition du site
et de l'utilisation du site.

Je peux voir plus de détails sur le modèle Présentation.
Donc, je vais sélectionner Détails.
J'aime l'apparence de la mise en page Présentation,
je vais donc sélectionner et cliquer sur Utiliser un modèle.
J'ai créé un site de communication à l'aide d'un modèle.
Je vais ajouter du contenu à ce site
avant d'être prêt à le partager avec d'autres,
mais je suis sur un super départ.

Vous savez maintenant comment ajouter un site de communication dans SharePoint.
Bientôt, vous serez prêt à ajouter du contenu
et à partager votre site avec d'autres.
