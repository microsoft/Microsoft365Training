Les applications sont des programmes que votre ordinateur exécute,
dans lesquelles vous allez passer la plupart de votre temps
sur votre ordinateur.
En général, une application est n'importe quel logiciel
utilisé pour accomplir une tâche.
Alors, que cette tâche soit
d'écrire un papier, de composer de la musique,
d'éditer des photos numériques, de jouer à un jeu vidéo,
de naviguer sur le web
ou de vérifier si vous avez reçu du courrier électronique.
Il existe beaucoup d'applications pour d'innombrables fins.
Certaines applications sont préinstallées sur votre ordinateur,
comme le navigateur web, Microsoft Edge et le traitement de texte de base,
par exemple WordPad.

Et vous pouvez acquérir et utiliser d'autres applications
en les téléchargeant et en installant
à partir des sites web de l'éditeur de logiciels
ou par le biais de services, comme par exemple, Microsoft Store.
Il en va de même pour les applications que vous utilisez sur les appareils mobiles,
votre téléphone.
Souvent les applications mobiles sont spécialisées
pour une utilisation en déplacement, comme les applications de navigation
qui vous aident à trouver les directions vers différents endroits
ou les applications de gestion du temps
qui vous envoient des notifications en vous rappelant les réunions,
les rendez-vous.

Vous pouvez obtenir presque n'importe quel type d'applications
sur votre appareil mobile.
Pour la plupart, les applications mobiles sont plus simples et moins puissantes
que les applications de bureau qui sont des applications conçues
pour fonctionner sur des ordinateurs de bureau,
ainsi, des ordinateurs plus puissants.
Ils sont donc plus adaptés à l'exécution d'applications
avec plus de fonctionnalités, plus de complexité.
Cependant, vous constaterez que les développeurs d'application
sont et font à la fois des versions de bureau
et des versions pour mobiles.

Leurs applications fonctionnent alors souvent ensemble.
Par exemple, vous pouvez commencer à rédiger un document dans Microsoft Word
sur votre téléphone
pendant que vous faites la navette dans un train.
Et puis, continuer à y travailler à partir de la version de bureau
lorsque vous arrivez à la maison ou à votre bureau.
