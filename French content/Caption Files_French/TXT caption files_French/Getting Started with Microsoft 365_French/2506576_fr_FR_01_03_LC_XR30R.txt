Imaginez ceci,
vous venez de mettre à jour votre CV pour postuler à un nouvel emploi.
Tous les détails ont fière allure,
sauf qu'il y a un problème.
Votre CV est réparti sur deux pages
et la demande d'emploi demandent une page maximum.
Dans cette leçon, je vais vous montrer
comment utiliser les marges
pour adapter le contenu que vous voulez
dans l'espace dont vous avez besoin.
Pour commencer,
téléchargez l'exemple de CV disponible dans les fichiers d'exercices.

La première chose que vous devez faire
est d'ouvrir le CV.
Comme il s'agit d'un document si important,
assurez-vous de le lire attentivement
pour vous assurer qu'il contient toutes les informations nécessaires.
Cette mise en page a l'air bien.
Mais il y a un peu de texte sur la deuxième page.
Donc, pour tout garder sur une seule page sans rien supprimer,
vous voudrez personnaliser vos marges,
Je vais vous montrer comment en accédant au ruban
et en sélectionnant la mise en page.

C'est là que Word affiche plusieurs options
pour la disposition du texte sur la page.
Vous pouvez explorer toutes ces fonctionnalités par vous-même,
mais pour l'instant,
sélectionnez simplement les marges.
À partir de là,
vous pouvez choisir des marges prédéfinies
ou des marges personnalisées.
Je vais choisir les marges personnalisées
pour me donner plus de flexibilité.
Lorsque je sélectionne des marges personnalisées,
une fenêtre apparaît
indiquant les marges actuelles.

Plus vos marges sont grandes,
moins vous aurez de place pour le texte sur une page.
Mais les réduire vous permettra d'insérer
plus de texte dans le même espace.
comme je dois faire tenir tout le texte sur une seule feuille de papier,
ses marges doivent être plus petites.
Je vais réduire les marges à un demi-pouce.
Sélectionnez OK
pour appliquer les marges.
Mise à jour, à votre document.
Votre CV tient désormais sur une seule page.

Si vous devez à nouveau ajuster les marges,
vous pouvez toujours revenir aux marges personnalisées
et apporter les modifications dont vous avez besoin.
Vous savez maintenant exactement
comment ajuster les marges dans un document Word
et obtenir tout ce dont vous avez besoin
sur la même page,
vous êtes prêt à reformater votre CV
ou tout autre document.
