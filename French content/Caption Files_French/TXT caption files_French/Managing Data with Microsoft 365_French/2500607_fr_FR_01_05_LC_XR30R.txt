Une fois que j'ai créé un sondage, j'ai hâte de l'envoyer à mes
clients et collaborateurs pour qu'ils puissent le remplir.
Dans cette leçon, je vais vous montrer comment
partager une enquête créée dans Microsoft Forms.
Dans le coin supérieur gauche d'office.com,
je sélectionne le lanceur d'applications et j'ouvre Forms.
À partir de là, j'ouvrirai le formulaire
Intérêt pour le conseil en planification de mariage.

Pour partager l'enquête et recueillir des données auprès
de mes clients,
je sélectionne Partager, dans le coin supérieur droit de l'écran.
Examinons ensemble le panneau Envoyer et recueillir les réponses.
Il existe quatre options pour partager un formulaire :
un lien, un courriel, un code QR ou une intégration.
Cette boîte de dialogue utilise par défaut l'option de liens,
je vais donc vous montrer cette option en premier.

Vous pouvez utiliser un lien dans un courriel,
une conversation Teams, ou même sur les réseaux sociaux.
Avant d'envoyer un lien, je vous recommande de cocher
la case Raccourcir l'URL,
cela générera une adresse web beaucoup plus courte.
Testons le lien avant de l'envoyer au client.
Tout d'abord, copiez l'URL raccourcie en sélectionnant
Copier.
Ouvrez ensuite un onglet vide dans votre navigateur,
puis cliquez sur le bouton droit dans la barre d'adresse et
sélectionnez Coller.

Vous voyez ainsi exactement comment le client verra le lien
dans son navigateur web.
Continuons à explorer.
L'option suivante consiste à créer un code QR.
Lorsque vous sélectionnez cette option,
Forms vous fournit une image de code QR
que vous pouvez alors télécharger et que vous pourrez ensuite
ajouter à n'importe quel document.
La troisième option consiste à obtenir un code pour intégrer
votre sondage dans vos sites web.

Vous pouvez utiliser le code HTML fourni par Forms, dans un site
web ou dans un courriel.
Pour obtenir le code, sélectionnez Copier.
La dernière façon de partager une enquête est par courriel.
Lorsque vous sélectionnez cette option,
le programme de messagerie par défaut de votre ordinateur ouvre
un nouveau courriel avec le lien du formulaire déjà ajouté dans
le corps du message.
Donc, une fois que votre enquête sera publiée,
je suis sûr que vous aurez hâte de voir les résultats. Pour voir les
résultats de mon enquête,
je sélectionne l'onglet Réponses pour afficher les données.

Dans la vue Réponses,
je peux ouvrir dans Excel pour afficher les données dans Excel.
Un fichier Excel contenant alors les résultats actuels de votre
enquête sera téléchargé, ce qui est utile pour l'analyse,
le partage et la création de rapports.
Pouvoir partager votre travail et obtenir les informations que vous
recherchez est facile lorsque vous utilisez Microsoft Forms.
Vous disposez maintenant d'une variété d'options pour envoyer
votre sondage aux clients actuels et potentiels.
