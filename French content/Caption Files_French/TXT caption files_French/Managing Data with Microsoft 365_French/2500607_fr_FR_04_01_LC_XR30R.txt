Les données sont devenues la base sur laquelle chacun,
dans le monde entier, prend des décisions.
Mais, aussi utiles que puissent être les données,
elles peuvent parfois nous sembler envahissantes.
Microsoft Power BI est un outil puissant de visualisation de
données, qui peut créer des rapports et des tableaux de bord
vous permettant de raconter une histoire à partir de vos données.
Dans cette leçon, je vais partager les avantages de
l'utilisation de Power BI, afin que vous puissiez approfondir vos
compétences en visualisation de données.

Microsoft Power BI fournit une infrastructure visuelle pour
partager les données.
Power BI peut aider à modéliser les informations conduisant à de
beaux rapports intuitifs et interactifs et,
dans votre public,
à obtenir des informations.
La fonctionnalité préférée de Power BI est qu'il est gratuit.
Power BI Desktop est téléchargeable gratuitement sur
votre PC à partir du Microsoft Store.

Power BI se connecte directement à vos données
Microsoft 365 existantes, stockées dans votre OneDrive ou SharePoint.
Pouvoir visualiser vos données vous donnera un avantage lorsque
vous communiquerez avec votre équipe et vos clients.
Je vous encourage à télécharger Power BI et à l'essayer.
