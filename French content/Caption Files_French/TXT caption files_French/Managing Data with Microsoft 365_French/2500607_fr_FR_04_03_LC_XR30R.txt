Les rapports ont besoin de données.
Avant de pouvoir créer des visualisations et obtenir
des informations, vous devez être en mesure
d'importer des ensembles de données dans Power BI.
Dans cette leçon, je vais vous expliquer deux façons
d'importer des données dans Power BI.
Vous devez installer Power BI sur votre PC en accédant au Microsoft
Store et en recherchant Power BI.
Lorsque vous ouvrez Power BI Desktop,
vous verrez l'écran de démarrage.

Je vais continuer et le fermer.
La première façon d'importer les données pour vos rapports
est à partir d'une feuille de calcul Excel.
Dans cet exemple, j'importe une liste de
clients depuis Excel.
Il y a deux façons de faire cela.
Vous pouvez soit sélectionner Classeur Excel dans le ruban
en haut de l'écran,
soit
Obtenir les données
et sélectionner Classeur Excel.

Je vais sélectionner Classeur Excel dans le ruban d'accueil.
Cela ouvre une boîte de dialogue, dans laquelle je peux sélectionner
la feuille de calcul que je souhaite importer.
Je vais ouvrir le fichier Client.
Vous pouvez le télécharger dans les fichiers d'exercice.
C'est ce qu'on appelle les données clients.
Le navigateur s'ouvre et a identifié une table de données
dans ma feuille de calcul, appelée Client.

Les données semblent correctes.
Je vais donc cocher la case devant Client,
puis cliquer sur Charger pour les importer dans Power BI.
La table Client s'affiche désormais dans le volet Champs.
Lorsque je développe la table Client à l'aide de la
liste déroulante,
je peux voir les champs qu'elle contient, avec un aperçu détaillé
de votre ensemble de données.
Vous pouvez sélectionner l'onglet Données dans le panneau de gauche.

La prochaine façon d'importer des données pour vos rapports
est à partir du web.
Disons que je veuille comparer mes ventes à la population
par département.
Je peux importer les données en ligne, directement dans Power BI.
Je vais commencer par sélectionner le menu déroulant sous
Accueil,
Obtenir les données,
puis sélectionner Web.

Ensuite, je vais coller l'URL de mes données et sélectionner OK.
Je sélectionne
Se connecter.
La boîte de dialogue Navigateur affiche toutes les tables Power
BI trouvées sur cette page web.

Je peux visualiser chacune d'elles pour trouver le tableau
que je recherche.
Je sélectionnerai
Liste des départements français, classés par population
décroissante au 1ᵉʳ janvier
mais, en fonction de l'aperçu, je devrai peut-être filtrer
certains champs et nettoyer les lignes.
Je sélectionne donc
Transformer les données, pour l'ouvrir directement
dans l'éditeur Power Query.

C'est ici que vous pourrez modifier les données utilisées
dans vos visualisations.
Pour ce faire, je vais supprimer les colonnes qui ne
m'intéressent pas.
Je n'ai pas besoin des colonnes
Rang 2022 ni Code Insee donc
je vais sélectionner la première colonne
et, dans le ruban supérieur, sélectionner Supprimer
les colonnes.

Je vais cliquer deux fois pour supprimer également
Code Insee.
Remarquez comment, au fur et à mesure que des
modifications sont apportées, vous pouvez les voir dans le
panneau Paramètres d'une requête à droite, dans la zone
Étapes appliquées.
Si j'ai besoin d'annuler une étape,
par exemple de supprimer une colonne,
je peux tout simplement sélectionner le X à côté
de cette étape.
J'ai donc maintenant une table nettoyée.

Elle est prête pour mon espace de travail.
Avant de fermer la table,
je vais lui donner un nom, en cliquant avec le bouton droit
sur le nom de la table et en sélectionnant Renommer.
Je vais entrer Département.
Ensuite, je sélectionnerai Fermer et appliquer pour enregistrer
ces modifications.

Maintenant, la table Département se trouve dans le panneau Champs,
sur le côté droit de l'écran.
Elle est prête à être utilisée.
La création de représentations visuelles commence par avoir
les bonnes données.
Réfléchissez aux données auxquelles vous avez accès et
essayez de les importer dans Power BI.
Qu'il s'agisse d'un classeur Excel ou du web,
Power BI peut vous aider à collecter vos données pour créer
des visualisations convaincantes.
