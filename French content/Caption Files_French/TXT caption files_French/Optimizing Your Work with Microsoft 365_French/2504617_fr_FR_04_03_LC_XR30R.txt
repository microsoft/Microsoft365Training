Faire référence à des ressources spécifiques dans un document
peut prendre beaucoup de place.
Avec les notes de bas de page,
vous pouvez garder vos références organisées et faciles à lire.
Dans cette leçon,
je vais vous montrer comment référencer votre contenu dans Word
avec des notes de bas de page,
et ajouter une table des matières.
Les notes de bas de page sont utilisées pour expliquer,
commenter ou fournir des références à quelque chose dans un document.

Une table des matières vous aide à localiser les informations spécifiques
dans un document.
Pour commencer, je vais ouvrir Word
à partir de la page d'accueil de Microsoft 365.
Vous pouvez suivre l'exemple de document Word
dans vos fichiers d'exercices.
C'est le document appelé Document d'événements produit client.
Dans cet exemple, je vais modifier un document produit
pour préparer un événement client que j'organise.

Tout d'abord, je vais ajouter des notes de bas de page.
Les notes de bas de page sont des citations
qui apparaissent au pied de page ou au bas de la page.
Une fois qu'une note de bas de page a été ajoutée,
vous aurez un petit numéro en exposant après le texte cité,
et la citation complète sur le pied de page
de cette page.
Pour insérer une note de bas de page,
je placerai mon curseur à droite du texte que je veux citer.

À partir de là, je vais accéder au ruban supérieur
et sélectionner Références.
À partir de là, je vais accéder au ruban supérieur.
sélectionnez Références,
puis sélectionner Insérer une note de bas de page.
À ce stade, une fenêtre apparaît
dans laquelle je peux spécifier les détails
de l'œuvre originale que je cite.
Je vais entrer :
« St. Claire Housen.

"Être un boss". Chapitre 3, page 46, ligne 22. »
C'est la première note de bas de page que j'ajoute au document.
Donc, le numéro un est dans la petite case.
Appuyez sur la touche Échap de votre clavier
ou sélectionnez le corps de votre document
pour enregistrer et fermer la note de bas de page.

Lorsque je suis en mode Édition,
je ne peux pas voir le texte de la citation au bas de page,
mais le numéro de la note de bas de page sera là.
Si je sélectionne le numéro,
je peux modifier les détails de la note de bas de page.
Pour voir comment la note de bas de page apparaît dans votre document,
je dois être en mode Lecture.
Accédez au ruban supérieur,
sélectionnez Affichage,
puis sélectionnez Mode Lecture.

Maintenant, lorsque je fais défiler le document,
je peux voir que les notes de bas de page sont en bas de la page.
Maintenant, je vais sélectionner l'onglet Modifier le document
dans le ruban supérieur,
puis je vais cliquer sur Modifier.
Word vous ramène alors en mode Édition
où vous pouvez continuer à éditer votre document.
Si une note de bas de page
n'est pas la méthode de citation que vous souhaitez utiliser,
envisagez une note de fin qui ajoute le texte de pied de page
à la fin du document au lieu de le mettre au bas de la page.

Accédez alors au ruban supérieur,
sélectionnez Références,
puis Insérer une note de fin.
Une autre façon de garder votre document organisé
est d'ajouter une table des matières.
Mais avant de créer une table des matières,
vous devez sélectionner votre texte et y appliquer un titre.
Word utilise les titres de votre document
pour créer une table des matières automatique
qui peut être mise à jour lorsque vous modifiez le texte,
la séquence ou le niveau d'un titre.

Mettez en surbrillance le texte auquel vous souhaitez appliquer un titre.
Ensuite, accédez à l'onglet Accueil,
et dans la galerie des styles, sélectionnez le style de titre
et le texte prend automatiquement cette forme de style.
Si vous avez des sous-titres,
sélectionnez un texte différent que vous aimeriez alors voir
sous le premier titre,
et appliquez-lui un style différent.

Vous êtes maintenant prêt à créer et à ajouter votre table des matières.
Placez maintenant votre curseur
à l'endroit où vous souhaitez insérer la table des matières.
Je la place généralement vers le début du document
ou après une page de titre.
Dans le ruban supérieur, sélectionnez Références.
Ensuite, sélectionnez Table des matières.

Dans le menu déroulant, choisissez Insérer une table des matières.
La table des matières est insérée, montrant les titres
et la numérotation des pages dans votre document.
Si vous apportez des modifications à votre document
qui affectent la table des matières,
vous pouvez la mettre à jour en sélectionnant l'option Mise à jour
qui apparaît au-dessus de la table des matières
lorsque vous la sélectionnez.

L'ajout de références sous forme de notes de bas de page
et la création d'une table des matières
est un excellent moyen d'améliorer vos documents.
Essayez d'ajouter ces références à votre prochain document.
